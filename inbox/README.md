# inbox/ — memory capture staging

This is the staging area for raw, uncurated memory. Drop quick notes, links, or
facts here as `.md` files; a curation pass (e.g. the `triage-inbox` workflow) later
sorts them into the right memory tier (`global/`, `workspaces/`, or a project's
`memory/`).

- Files here are **transient** — they are gitignored (only this README and a
  `.gitkeep` are tracked). Don't rely on anything in `inbox/` surviving curation.
- Think of it as an outbox-for-your-brain: capture now, organize later, never lose
  a fact because there wasn't a "right place" for it yet.
